# 更新日志

## 2026-01-17 更新 v2

### 端口变更
- 后端默认端口：**3077**
- 前端默认端口（Docker）：**3088** (之前是80，避免端口占用)
- 所有相关配置文件已同步更新

### Docker 部署端口说明
- 后端 API：`http://localhost:3077`
- 前端应用：`http://localhost:3088`
- 容器内部前端仍使用 80 端口，通过端口映射到主机的 3088

### 新增功能
- ✨ 新增 **API使用教程** 页面
  - 路径: `/api-docs`
  - 提供详细的API调用示例
  - 支持多种编程语言示例代码

### API文档包含内容

**代码示例：**
1. 命令行 (cURL)
2. HTML 无样式版本
3. HTML 带样式版本（精美渐变设计）
4. JavaScript (Fetch & Axios)
5. Python (requests库)
6. Java (HttpClient)
7. PHP (cURL)
8. Go
9. C# (.NET)
10. Ruby

**其他内容：**
- 快速开始指南
- API端点说明
- 响应格式示例
- 使用场景说明
- 注意事项

### 页面特性
- 支持动态修改仓库名，实时更新示例代码
- 一键复制代码功能
- 简洁现代的UI设计
- 完全响应式布局

### 如何访问

部署后访问以下路径查看API文档：
```
http://localhost:3088/api-docs
```

或在导航栏点击 "API文档" 链接。

---

## 使用示例

访问文档页面后，您可以：
1. 修改仓库名输入框，自动更新所有示例代码
2. 点击"复制代码"按钮快速复制示例
3. 查看不同编程语言的实现方式
4. 了解API的完整使用方法

## 部署注意事项

如果您已经部署了旧版本，需要重新构建容器：

```bash
docker-compose down
docker-compose up -d --build
```

本地开发无需特殊操作，重启服务即可。
